# 로컬 웹서버 실행방법

.\Scripts\activate
^ 위 명령어 실행해서 파이썬 가상환경 ON

python ./app.py
^ 가상환경 실행 후 위 명령어로 플라스크 웹 서버 실행

로컬에서 실행했을 시, 'http://127.0.0.1:5000' 주소에서 웹 서비스 실행됨

deactivate
^ 위 명령어로 가상환경 종료
---

# 프로젝트 경로 설명

1. 기본적인 플라스크 페이지와 렌더링 방법들을 담아둠
2. input_tag기능의 경우, 블루프린트 사용법을 보여주기 위해 WVA에서 떼어옴 'http://127.0.0.1:5000/input_tag' 에서 기능동작 확인 가능
3. 'input_tag_parser.py'가 기능역할, 'input_tag_analysis.py'가 연결해서 불러와주는 역할을 함.
